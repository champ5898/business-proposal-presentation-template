# Business Proposal Presentation Template

A modern, single-page business proposal presentation built with React and Tailwind CSS.

## Features

- Single-page scrolling layout with all slides
- Print-friendly with page breaks for PDF export
- Professional design with gradients and animations
- Fully customizable content
- Contact information: info@daslabs.online, www.daslabs.online

## Running the code

```bash
npm install
npm run dev
```

## Building for production

```bash
npm run build
```

## Exporting to PDF

1. Open the app in your browser
2. Press Ctrl+P (or Cmd+P on Mac)
3. Select "Save as PDF" as the destination
4. Set orientation to Landscape
5. Enable "Background graphics" to preserve colors
6. Save the PDF
